## What Atelier does

Atelier is a **visual customization toolkit for Confluence Cloud**. Eight macros let you create beautifully styled pages with branded colors, patterns, and rich formatting — **no custom CSS, no HTML**, no copy-pasted hacks. Pages stay native to Confluence and export cleanly to PDF and Word.

## The 8 macros

| Macro | Purpose | Highlights |
| --- | --- | --- |
| **Background** | Full section with pattern fill | Patterns, page embedding, rich text, text shadow, custom height |
| **Banner** | Decorative header section | Patterns, height presets, centered text, font size control |
| **Text Color** | Apply custom text color | Color picker, font family, alignment, shadow options |
| **Divider** | Horizontal separator | Solid, dashed, dotted, gradient, and pattern fill styles |
| **Callout** | Styled info / alert box | 13 icon types, 7 border styles, accent colors, custom titles |
| **Button** | Call-to-action link | 4 shapes, pattern fill, emoji, URL validation, multiple sizes |
| **Spacer** | Vertical whitespace | Adjustable height, optional background color or pattern |
| **Card** | Content card with CTA | Patterned header, body text, call-to-action link, corner styles |

## Why Atelier

**"Our pages look like every other Confluence wiki."**  
64 background patterns and per-space color palettes turn generic pages into branded surfaces — without anyone editing CSS.

**"We can't keep our docs visually consistent."**  
Theme presets save pattern-and-color combinations once, reusable across every macro and every page.

**"My PDF exports lose all the styling."**  
Atelier's macros export cleanly for both Confluence's PDF and Word downloads.

**"We need brand colors but can't deploy a Confluence theme."**  
Space-level color palettes give brand teams a controlled surface — set it once at the space level, every author can pick from the palette in any macro.

**"Embedded content from other pages is clunky."**  
The Background macro embeds content from other Confluence pages with full permissions support — viewers only see content they have access to.

## Key features

- **64 background patterns** — Gradients, geometric, organic, and technical, ready to drop into any macro.
- **Rich text editor** — Bold, italic, underline, lists, links, emoji, highlighting, and font sizes.
- **Space color palettes** — Define brand colors per space; available in every macro automatically.
- **Reusable theme presets** — Save pattern + color combinations for one-click macro setup.
- **Font management** — Curate suggested fonts per space for consistent typography.
- **Page embedding** — Embed content from other Confluence pages with full permissions support.
- **PDF / Word export** — All macros export cleanly for Confluence's PDF and Word downloads.

## Space settings

Space administrators can access **Atelier Settings** under *Space Settings → Integrations* to manage:

- **Color palettes** — Up to 50 brand colors shared across all macros.
- **Theme presets** — Up to 50 saved pattern + color combinations.
- **Suggested fonts** — Up to 30 curated fonts for consistent styling.

## Platform & security

- Built on **Atlassian Forge** — runs entirely within Atlassian Cloud infrastructure.
- **No external servers** — your data never leaves Atlassian.
- **Permission-aware** — embedded content respects Confluence permissions; users only see what they have access to.
- **Encrypted storage** — all space settings stored securely using Forge's encrypted storage.

## Getting started

1. Install Atelier from the Atlassian Marketplace.
2. Open any Confluence page in edit mode.
3. Type `/atelier` to see the available macros, or pick one from the macro list.
4. Configure colors, patterns, and content with the inline editor.
5. (Space admins) Visit **Space Settings → Integrations → Atelier** to set up palettes, presets, and fonts.
